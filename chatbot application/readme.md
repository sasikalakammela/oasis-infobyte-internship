# Simple Chat Application (Python)

## Project Overview
Developed a client-server based chat application using Python that enables real-time two-way communication between users via terminal. The project demonstrates core networking concepts and concurrent message handling.

## Key Features
* Real-time bidirectional communication
* Client-server architecture using sockets
* Concurrent message handling using threading
* Lightweight and command-line based implementation
* Error handling for connection issues

## Technical Skills Demonstrated
* Python Programming
* Socket Programming (TCP/IP)
* Multithreading
* Network Communication
* Debugging and Error Handling

## Tools and Technologies
* Python 3.x
* Socket Library
* Threading Module
